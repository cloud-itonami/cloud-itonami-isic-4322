# Operator Guide

## First Deployment
1. Register operator, contracts/sites, trade licenses, staff and
   robots.
2. Import existing permit and inspection history.
3. Run read-only license-scope and installation/inspection robot
   mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run inspection/commissioning record and audit export.

## Minimum Production Controls
- licensed-tradesperson-scope validation before any installation
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (commissioning
  without inspection, gas-line/pressure-system work outside license
  scope)
- evidence-backed inspection/commissioning records
- audit export for every dispatch, sign-off and record
- backup manual permit-review process

## Certification
Certified operators must prove robot-safety integrity, license-scope
discipline, evidence-backed inspection records and human review for
commissioning-affecting actions.
